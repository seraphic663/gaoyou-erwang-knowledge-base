# 高邮二王知识库（项目总览）

本仓库采用“资料、方案、网站、参考”分层结构，便于申报、开发与答辩并行推进。

## 目录结构

- 01-项目申报：申报书、意向表等正式提交材料
- 02-研究方案：研究设计与任务规划（含 `task.md`）
- 03-项目网站：可运行 Demo 网站（前后端一体）
- 04-参考资料：学校通知、指南与相关附件
- data：Demo 数据文件
- media：图片等资源
- note：个人笔记

## 网站启动

在仓库根目录执行：

```bash
npm --prefix 03-项目网站 start
```

然后访问：

http://localhost:3000

## 可靠性说明（已加固）

- 网站可自动适配当前目录结构读取数据
- 增加 `/api/health` 健康检查接口
