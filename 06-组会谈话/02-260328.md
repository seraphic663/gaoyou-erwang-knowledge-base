# 02-260328 组会纪要

## 一、会议基本信息

- 会议主题：高邮二王大创小组组会
- 会议时间：2026年3月28日 16:00-18:00
- 会议地点：立德楼明德书院空间线下会议

## 二、会议议程

1. 明确项目研究目标
2. 讨论高邮二王“校读学”与数据库整理方向
3. 商议项目经费预算
4. 统一立项申报书写作要求

## 三、会议核心共识

### 1. 项目目标不能只停留在“结果整理”

本项目的重点不只是罗列二王的考据结论，还要进一步整理：

- 讹误、通假、训释等问题类型
- 考据表述背后的逻辑
- 不同论证模式与篇章结构
- 考据过程中各因素之间的权衡关系

会议认为，项目最终要呈现的不只是“二王说了什么”，更是“二王如何完成论证”。因此，数据库和平台都应兼顾结果层与过程层。

### 2. 高邮二王研究的整理单位应落实到“单条札记”

会议提出，应把每条札记视为一个基本分析单位，至少回答以下问题：

- 该条札记对应哪一部书
- 它处理的是哪一类问题
- 它解决的是哪个具体字词或句段
- 它如何展开论证

会议初步归纳出考据过程可拆为：

1. 发疑
2. 判断方向
3. 调用材料
4. 组织论证
5. 推广或归纳

这一思路对后续数据库中“案例”与“过程”字段的设置有直接指导意义。

### 3. 二王校读学的整理应覆盖“对象、方法、统计、引证”四层

会议讨论认为，后续整理不应只记录结论，而应同步关注以下内容：

- 正字、破字、训释等类型区分
- 每部书中讹误、通假等问题的数量与分布
- 字形、音韵等背景知识与具体语境的结合方式
- 校读用语、校读过程与常见表达
- 考据中所引例证、所引书目及其先后顺序
- 二王整体知识系统与单条文本之间的互动关系

换言之，本项目应同时处理“二王系统”与“具体文本”两个层面，不能只做零散案例摘录。

### 4. 项目主题可概括为“高邮二王的知识生产与数字化呈现”

会议形成一个较清晰的中间表述：

> 高邮二王的知识生产与数字化呈现。

这一提法有两个好处：

- 能把传统校读学与数字人文平台建设连接起来
- 能同时容纳文本整理、考据过程分析和可视化展示

### 5. 经费预算以必要项为主，保持克制

会议初步只提出两类预算方向：

- 劳务费
- 业务费

共识是：预算不宜发散，应优先服务于文献整理、数据处理、平台搭建与必要学术支撑。

### 6. 申报书写作应突出“材料、方法、成果形态”

会议对申报文本的要求是：

- 行文要简洁
- 要清楚说明立足什么材料
- 要说明运用什么方法
- 要说明最终形成什么成果

会议同时确认，本项目最值得强调的优势主要有三点：

1. 跨学科优势：人文研究与数字技术结合
2. 双重价值：兼具学术研究与文化传承转化意义
3. 材料优势：可对“王氏四种”做较系统的整理，并进一步揭示其内部逻辑关联

## 四、对项目落地的直接启发

结合本次讨论，后续工作宜重点落实以下方向：

1. 以“单条考据”为基础单位进行结构化整理
2. 区分问题类型、方法类型和证据类型
3. 同时保留结论、过程、引证顺序与统计信息
4. 在申报书中把“研究对象 - 方法路径 - 平台成果”写成一条清楚主线

## 五、会议结论

本次会议进一步明确：本项目的核心不只是做一个二王文献资料库，而是尝试把二王的知识生产过程转化为可检索、可分析、可展示的数字化对象。项目最终关心的根本问题可以概括为：

> 如何跨越时空理解文本。
